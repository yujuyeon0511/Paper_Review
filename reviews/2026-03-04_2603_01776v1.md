## 📝 초록 (Abstract)

Quantization is pivotal for mitigating the significant memory and computational overhead of Large Language Models (LLMs). While emerging transformation-based methods have successfully enhanced quantization by projecting feature spaces onto smoother manifolds using orthogonal matrices, they typically enforce a rigid one-to-one transformation constraint. This static approach fails to account for the dynamic patterns inherent in input activations, particularly within diffusion LLMs (dLLMs) and Multimodal LLMs (MLLMs), where varying token types exhibit distinct distributions. To advance this, we propose FreeAct, a novel quantization framework that relaxes the static one-to-one constraint to accommodate dynamic activation disparities. Theoretically, we leverage the rank-deficient nature of activations to derive a solution space that extends beyond simple inverse matrices, enabling the decoupling of activation transformations from weights. Methodologically, FreeAct identifies token-specific dynamics (i.e., vision v.s. text, or masked tokens) and allocates distinct transformation matrices to the activation side, while maintaining a unified, static transformation for the weights. Extensive experiments across dLLMs and MLLMs demonstrate that FreeAct significantly outperforms baselines, up to 5.3% performance improvement, with in-depth analyses. Our code will be publicly released.

## 🔍 AI 분석

## 논문 분석: FreeAct

### 📄 논문 요약
FreeAct 논문은 LLM 양자화의 메모리 및 계산 부담을 줄이기 위한 새로운 프레임워크를 제시합니다. 기존의 양자화 방법은 고정된 one-to-one 변환 제약을 사용했지만, diffusion LLM(dLLM) 및 Multimodal LLM(MLLM)의 동적인 활성화 패턴을 제대로 반영하지 못했습니다. FreeAct는 이러한 한계를 극복하기 위해 활성화 변환 제약을 완화하고, 활성화와 가중치의 변환을 분리하여 token별 동적 특성을 고려합니다. 실험 결과, dLLM 및 MLLM에서 기존 방법 대비 성능 향상을 보여주며, 최대 5.3%의 성능 개선을 달성했습니다.

### 🆕 새로운 점 (Novelty)
FreeAct의 핵심적인 새로운 점은 활성화 변환에 대한 고정된 one-to-one 변환 제약을 완화하고, token별 동적 특성을 고려하여 활성화 변환을 분리하는 방식입니다. 특히, diffusion LLM 및 MLLM의 활성화 패턴을 분석하여, 각 token 유형에 따라 다른 변환 행렬을 할당하는 방법을 제안함으로써 기존 양자화 방법의 한계를 극복했습니다. 또한, 활성화의 rank-deficient 특성을 활용하여 변환 공간을 확장하고, 단순한 역행렬을 넘어선 솔루션 공간을 탐색하는 접근 방식도 새로운 시도입니다.

### 💪 강점 (Strengths)
1. **동적 활성화 패턴 고려:** 기존 양자화 방법이 활성화의 동적인 특성을 제대로 반영하지 못하는 문제를 해결하기 위해, token별 동적 특성을 고려하여 활성화 변환을 분리하는 방식을 제안했습니다.
2. **활성화 변환 제약 완화:** 고정된 one-to-one 변환 제약을 완화하여, 활성화 변환의 유연성을 높이고 성능 향상을 가능하게 했습니다.
3. **실험적 검증:** dLLM 및 MLLM에서 기존 방법 대비 성능 향상을 입증하며, 제안하는 방법의 효과를 검증했습니다. 특히, 5.3%의 성능 개선은 괄목할 만한 성과입니다.

### ⚠️ 약점/한계점 (Limitations)
1. **token별 동적 특성 분석의 복잡성:** 각 token 유형에 맞는 변환 행렬을 결정하는 과정이 복잡할 수 있으며, 자동화된 방법론이 필요할 수 있습니다.
2. **활성화의 rank-deficient 특성 활용:** 활성화의 rank-deficient 특성을 활용하는 방법론이 아직 초기 단계이며, 더 많은 연구가 필요할 수 있습니다.
3. **LLM 종류에 대한 제한:** 실험 결과가 dLLM 및 MLLM에 국한되어 있으며, 다른 종류의 LLM에도 적용 가능성을 확인해야 합니다.

### 🔗 내 연구와의 연관성
FreeAct 논문의 활성화 변환 분리 방식은 제가 현재 진행하고 있는 Vision-Text Alignment 연구와 밀접하게 관련됩니다. 특히, Vision encoder와 Text LLM 간의 정렬 문제를 해결하기 위해, 각 modality에 맞는 변환 행렬을 사용하는 아이디어를 FreeAct에서 얻을 수 있습니다. 또한, MLLM의 다양한 token 유형(이미지, 텍스트 등)에 따라 다른 변환 행렬을 적용하는 방식은 문서/차트 VQA 연구에도 적용 가능할 것으로 예상됩니다.

### 💡 연구 아이디어 제안
1. **자동화된 token별 변환 행렬 결정:** 각 token 유형에 최적화된 변환 행렬을 자동으로 결정하는 알고리즘을 개발하여, FreeAct의 적용 범위를 넓힐 수 있습니다.
2. **활성화 변환 분리 방식의 확장:** FreeAct에서 제안하는 활성화 변환 분리 방식을 다른 양자화 방법(예: quantization-aware training)에 적용하여 성능 향상을 검증할 수 있습니다.
3. **한국어 MLLM에 대한 적용:** FreeAct를 한국어 MLLM에 적용하여, 한국어 데이터에 특화된 양자화 성능을 평가하고 개선할 수 있습니다. 특히, 한국어 텍스트와 이미지의 동적 특성을 고려하여 변환 행렬을 조정하는 연구가 필요합니다.
