## 📝 초록 (Abstract)

Quantization is pivotal for mitigating the significant memory and computational overhead of Large Language Models (LLMs). While emerging transformation-based methods have successfully enhanced quantization by projecting feature spaces onto smoother manifolds using orthogonal matrices, they typically enforce a rigid one-to-one transformation constraint. This static approach fails to account for the dynamic patterns inherent in input activations, particularly within diffusion LLMs (dLLMs) and Multimodal LLMs (MLLMs), where varying token types exhibit distinct distributions. To advance this, we propose FreeAct, a novel quantization framework that relaxes the static one-to-one constraint to accommodate dynamic activation disparities. Theoretically, we leverage the rank-deficient nature of activations to derive a solution space that extends beyond simple inverse matrices, enabling the decoupling of activation transformations from weights. Methodologically, FreeAct identifies token-specific dynamics (i.e., vision v.s. text, or masked tokens) and allocates distinct transformation matrices to the activation side, while maintaining a unified, static transformation for the weights. Extensive experiments across dLLMs and MLLMs demonstrate that FreeAct significantly outperforms baselines, up to 5.3% performance improvement, with in-depth analyses. Our code will be publicly released.

## 🔍 AI 분석

## 📄 논문 요약
본 논문은 LLM 양자화의 성능을 향상시키기 위한 새로운 프레임워크인 FreeAct를 제안합니다. 기존 양자화 방법들이 활성화 공간을 매끄럽게 만들기 위해 정적인 일대일 변환 제약을 가하는 반면, FreeAct는 이러한 제약을 완화하여 입력 활성화의 동적인 패턴을 고려합니다. 특히, diffusion LLM (dLLM) 및 Multimodal LLM (MLLM)에서 토큰 유형에 따라 다른 분포를 보이는 활성화 패턴에 주목하여, 토큰별 동적 변환 행렬을 적용합니다. 실험 결과, FreeAct는 기존 방법 대비 최대 5.3%의 성능 향상을 보였습니다.

### 🆕 새로운 점 (Novelty)
FreeAct는 LLM 양자화에서 기존의 정적인 일대일 변환 제약을 완화하고, 활성화의 동적인 특성을 고려하는 새로운 접근 방식을 제시합니다. 특히, MLLM에서 텍스트와 비전 토큰의 분포 차이를 인지하고 각각 다른 변환 행렬을 적용하는 아이디어가 독창적입니다. 활성화 변환과 가중치 변환을 분리하여 활성화 측면에 더 유연성을 부여하는 방식 또한 새로운 시도입니다.

### 💪 강점 (Strengths)
1. **MLLM에 대한 고려:** MLLM의 특성을 고려하여 텍스트와 비전 토큰의 활성화 분포 차이를 반영한 점이 돋보입니다. 이는 사용자의 연구 주제인 Korean MLLM 개발 및 평가에 직접적으로 관련될 수 있습니다.
2. **성능 향상:** dLLM 및 MLLM에서 기존 방법 대비 상당한 성능 향상을 보여주며, 실용적인 양자화 솔루션으로서의 가능성을 입증했습니다.
3. **이론적 근거:** 활성화의 rank-deficient 특성을 활용하여 변환 행렬을 설계하는 이론적 근거를 제시하여, 방법론의 타당성을 뒷받침합니다.

### ⚠️ 약점/한계점 (Limitations)
1. **복잡도 증가:** 토큰별 동적 변환 행렬을 적용하는 방식은 기존 방법보다 복잡도를 증가시킬 수 있습니다. 이는 모델 추론 속도에 영향을 미칠 수 있습니다.
2. **토큰 분류 기준:** 텍스트와 비전 토큰 외 다른 토큰 유형에 대한 분류 기준 및 적용 방식에 대한 구체적인 설명이 부족합니다.
3. **한국어 MLLM에 대한 검증 부재:** 실험 결과가 영어 기반 MLLM에 한정되어 있어, 한국어 MLLM에 대한 성능 검증이 필요합니다.

### 🔗 내 연구와의 연관성
본 논문은 사용자의 연구와 매우 밀접하게 연관됩니다. 특히, 한국어 MLLM 개발 및 성능 향상에 기여할 수 있는 가능성이 높습니다. 사용자는 현재 Vision encoder와 Text LLM 간의 정렬(alignment) 연구를 진행하고 있는데, FreeAct는 이러한 정렬 과정에서 발생하는 활성화 분포의 차이를 완화하는 데 도움을 줄 수 있습니다. 또한, MLLM 모델 머징을 통한 성능 향상 연구에도 FreeAct를 적용하여 양자화된 모델의 성능을 더욱 개선할 수 있습니다.

### 💡 연구 아이디어 제안
1. **한국어 MLLM에 FreeAct 적용:** 한국어 MLLM에 FreeAct를 적용하여 성능 향상 효과를 검증하고, 한국어 특성에 맞는 토큰 분류 기준을 연구합니다.
2. **FreeAct와 Vision-Text Alignment 연구 결합:** FreeAct를 활용하여 Vision encoder와 Text LLM 간의 활성화 분포를 정렬하는 새로운 방법을 연구합니다.
3. **FreeAct를 활용한 MLLM 모델 머징:** FreeAct를 적용하여 양자화된 MLLM 모델들을 머징하여 성능을 극대화하는 방법을 연구합니다.
4. **FreeAct의 복잡도 감소 방안 연구:** FreeAct의 복잡도를 줄이면서 성능을 유지할 수 있는 방법을 연구하여 실용적인 양자화 솔루션으로 발전시킵니다.
